# AutoBikeLED

## コンセプト
- 既製品から取り出した回路を利用
- ヘッドライト・リアライト・サイコン照明を一つのシステムで駆動
  - リアライトは走行を検知して点灯
  - ヘッドライト・サイコン照明は走行中かつ暗いときに自動で点灯
